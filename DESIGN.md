# Pathly Design Guidelines

## Visual Identity
- Style: Clean, modern SaaS admin dashboard
- Feel: Premium, trustworthy, motivating
- Layout: Sidebar navigation + main content area

## Color Palette
- Primary: #1D4ED8 (rich blue — CTAs, active states, key actions)
- Secondary: #0F766E (teal — supporting accents, success states)
- Accent: #7C3AED (purple — highlights, badges, milestone celebrations)
- Background: #F8FAFC (near-white, airy)
- Foreground: #0F172A (deep navy text)
- Card: #FFFFFF
- Muted: #F1F5F9
- Border: #E2E8F0
- Destructive: #DC2626

## Typography
- Heading font: Plus Jakarta Sans — bold, modern, confident
- Body font: Inter — highly readable, clean
- Use font-weight variation to establish hierarchy (700 headings, 500 labels, 400 body)

## Layout Patterns
- Sidebar navigation: fixed left, collapsible, dark navy background
- Main content: generous whitespace, 24–32px padding
- Cards: rounded-xl, subtle shadow, white background
- Data tables: striped rows, sortable headers, pagination
- Forms: labeled inputs, helper text, inline validation
- Badges: rounded-full, color-coded by notification type

## Component Guidelines
- Buttons: rounded-lg, primary filled for main actions, outline for secondary
- Toggles: teal when active, muted when off
- Progress indicators: blue fill on muted track
- Notification type cards: icon + label + toggle + frequency selector
- Empty states: illustrated, encouraging copy

## Elevation & Depth
- Cards: shadow-sm on rest, shadow-md on hover
- Modals: shadow-xl with backdrop blur
- Sidebar: slight shadow-right to separate from content

## Responsiveness
- Desktop-first (admin dashboard context)
- Mobile-responsive with collapsible sidebar
