# Pathly — AI Career Coaching Platform

## What It Is
Pathly is an AI-driven proactive email coaching platform for job seekers. It monitors each user's career profile, activity, roadmap, and progress, then sends personalized email guidance at the right moment — without users having to ask.

## Target Users
- Primary persona: Job seekers mid-search
- Users who want an always-on career mentor guiding them through their job hunt

## Brand
- App name: Pathly
- Tagline: Your always-on career mentor
- Tone: Encouraging, professional, actionable

## Core Features

### AI-Generated Personalized Career Emails
Pathly automatically sends proactive coaching emails across 9 notification types:
1. Resume update reminders
2. Skill learning suggestions
3. Certification recommendations
4. Interview prep reminders
5. Mock interview invitations
6. Weekly/monthly progress summaries
7. Job application nudges/reminders
8. ATS optimization tips
9. Milestone celebrations and career tips

### Notification Preferences Page
Users control exactly what they receive:
- Toggle each of the 9 notification types on/off
- Set frequency per type: Instant / Daily / Weekly / Monthly
- Set preferred delivery time (e.g., 9:00 AM)
- Set preferred days (e.g., Monday, Wednesday, Friday)

### Admin Dashboard
- User management with data tables
- Notification analytics (delivery rates, open rates per type)
- System configuration for AI coaching parameters
- RBAC: Admin, Coach, User roles
